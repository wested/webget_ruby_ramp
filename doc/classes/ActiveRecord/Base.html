create_or_update(options = {})

Create a record, or update a record if value passed matches a field in the AR object; includes method_missing function to make code more readable

Most common use will be for testing (fixture/mock object generation)

Three versions of method included: create_or_update create_or_update_by create_or_update_by_xxx (where xxx is a field name)

Inspired by www.intridea.com/2008/2/19/activerecord-base-create_or_update-on-steroids-2

Example

{ “admin” => [“Administrator”, 1000], 

  "member"    => ["Member", 1],
  "moderator" => ["Moderator", 100],
  "disabled"  => ["Disabled User", -1] }.each_pair do |key, val|
    Role.create_or_update_by_key(:key => key, :name => val[0], :value => val[1])

end

create_or_update_by(condition_keys, attribute_pairs = {})

Create or update a record by field (or fields). This will look for each field name as a key in the options hash.

Example

  attributes={:name="John Smith", :email=>"john@example.com", :birthdate=>'1980/01/01'}
  User.create_or_update_by(:email,attributes)
  => if a user with that email exists then update his name and birthdate, else create him

Example with multiple conditions

  attributes={:name="John Smith", :email=>"john@example.com", :birthdate=>'1980/01/01'}
  User.create_or_update_by([:name,:birthdate],attributes)
  => if a user with that name and birthdate exists then update his email, else create him

The fields can be any mix of symbols or strings. The option keys can be any mix of symbols or strings.

method_missing_with_create_or_update(method_name, *args)

Method missing for create_or_update_by_xxx that forwards to create_or_update_by

Example

  MyModel.create_or_update_by_foo(options)
  => MyModel.create_or_update_by(:foo,option)

seed(condition_keys, attribute_pairs = {})

Set up a database with initial data, e.g. in rake db:seed method.

This will look for each field name as a key in the options hash.

This method calls create_or_update_by (and you may want to change this behavior to do more, e.g. to test that a DB and table exists).
