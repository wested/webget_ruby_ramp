                        <div id="pivot-method" class="method-detail ">
                                <a name="M000132"></a>

                                <div class="method-heading">

                                        <span class="method-name">pivot</span><span
                                                class="method-args">(direction='keys',&b)</span>
                                        <span class="method-click-advice">click to toggle source</span>

                                </div>

                                <div class="method-description">

                                        <p>

Hash#pivot aggregates values for a hash of hashes, for example to calculate subtotals and groups.

Example: 



  h={
   &quot;a&quot;=&gt;{&quot;x&quot;=&gt;1,&quot;y&quot;=&gt;2,&quot;z&quot;=&gt;3},
   &quot;b&quot;=&gt;{&quot;x&quot;=&gt;4,&quot;y&quot;=&gt;5,&quot;z&quot;=&gt;6},
   &quot;c&quot;=&gt;{&quot;x&quot;=&gt;7,&quot;y&quot;=&gt;8,&quot;z&quot;=&gt;9},
  }
  h.pivot(:keys) =&gt; {&quot;a&quot;=&gt;[1,2,3],&quot;b&quot;=&gt;[4,5,6],&quot;c&quot;=&gt;[7,8,9]}
  h.pivot(:vals) =&gt; {&quot;x&quot;=&gt;[1,4,7],&quot;y&quot;=&gt;[2,5,8],&quot;z&quot;=&gt;[3,6,9]}

Calculating subtotals

The pivot method is especially useful for calculating subtotals.

Example



  r = h.pivot(:keys)
  r['a'].sum =&gt; 6
  r['b'].sum =&gt; 15
  r['c'].sum =&gt; 24

Example



  r=h.pivot(:vals)
  r['x'].sum =&gt; 12
  r['y'].sum =&gt; 15
  r['z'].sum =&gt; 18

Block customization

You can provide a block that will be called for the pivot items.

Examples



  h.pivot(:keys){|items| items.max } =&gt; {&quot;a&quot;=&gt;3,&quot;b&quot;=&gt;6,&quot;c&quot;=&gt;9}
  h.pivot(:keys){|items| items.join(&quot;/&quot;) } =&gt; {&quot;a&quot;=&gt;&quot;1/2/3&quot;,&quot;b&quot;=&gt;&quot;4/5/6&quot;,&quot;c&quot;=&gt;&quot;7/8/9&quot;}
  h.pivot(:keys){|items| items.inject{|sum,x| sum+=x } } =&gt; {&quot;a&quot;=&gt;6,&quot;b&quot;=&gt;15,&quot;c&quot;=&gt;24}

Examples



  h.pivot(:vals){|items| items.max } =&gt; {&quot;a&quot;=&gt;7,&quot;b&quot;=&gt;8,&quot;c&quot;=&gt;9}
  h.pivot(:vals){|items| items.join(&quot;-&quot;) } =&gt; {&quot;a&quot;=&gt;&quot;1-4-7&quot;,&quot;b&quot;=&gt;&quot;2-5-8&quot;,&quot;c&quot;=&gt;&quot;3-6-9&quot;}
  h.pivot(:vals){|items| items.inject{|sum,x| sum+=x } } =&gt; {&quot;a&quot;=&gt;12,&quot;b&quot;=&gt;15,&quot;c&quot;=&gt;18}
